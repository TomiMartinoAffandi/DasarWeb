position static

position static untuk mengatur element menjadi statis secara default. elemen akan mengikuti posisi normal secara default, elemen tidak dipengaruhi oleh properti seperti top, bottom, left dan right.:

elemen ini diatur menggunakan posisi: static;




position relative:

sebuah elemen html yang menggunakan posisi relativ akan terletak pada posisi normal. mengatur properti atas, bawah, kiri dan kanan dari elemen yang diposisikan "relatif" akan membuat jauh dari posisi normalnya. konten lain tidak akan disesuaikan agar sesuai dengan celah yang ditinggalkan oleh elemen tersebut

elemen ini diatur menggunakan posisi: relative;



position fixed

sebuah elemen html yang disetting dengan position fixed akan memiliki sifat tetap, tanpa ada perubahan bahkan jika halaman web di scroll, berlaku pengaturan atas, bawah, kiri dan kanan pada position fixed

elemen ini diatur menggunakan posisi fixed




















posisi absolut

elemen html yang menggunakan posisi absolut akan diposisikan relative dengan elemen lain yang mendahuluinya yang terdekat, bukan relatif terhadap layar secara normal

elemen ini diatur menggunaka posisi: relative;
elemen ini diatur menggunakan posisi: absolut;
